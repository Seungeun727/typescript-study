# TypeScript 

## 🔦 Description 

## ⚙️ 프로젝트 설치

## npm install 

```
  npm init -y
```

## typeScript 패키지 설치

```
 npm install -D typescript
```

## 실행 : node_modules => typescript

- npx tsc

- node_modules./bin/tsc
- node_modules/typescript/bin/tsc

## tsconfig.json 파일 생성

```
npx tsc --init
```

## 파일 생성  

```
nano [파일이름명.ts]
```

## 파일 변경 내용 감지 

``` 
tsc -watch 
```

## build

```
npm run build
```
